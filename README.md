# musicplayer-MVC
Uma landing page responsiva e moderna para músicos com player de música integrado, desenvolvida em PHP seguindo o padrão MVC (Model-View-Controller).
Esta landing page foi desenvolvida para músicos que desejam compartilhar suas músicas de forma profissional e elegante. O projeto inclui um player de música completo com controles semelhantes aos de um painel automotivo, oferecendo uma experiência intuitiva e moderna aos visitantes.
