# Music player-MVC
Uma landing page responsiva e moderna para músicos com player de música integrado, desenvolvida em PHP seguindo o padrão MVC (Model-View-Controller).
Esta landing page foi desenvolvida para músicos que desejam compartilhar suas músicas de forma profissional e elegante. O projeto inclui um player de música completo com controles semelhantes aos de um painel automotivo, oferecendo uma experiência intuitiva e moderna aos visitantes.
<br>
Características Principais:
.Player de Música Completo - Play, pause, stop, próxima, anterior.
.Barra de Progresso Interativa - Clique para navegar pela música.
.Controle de Volume com Slider ajustável.
.Design Responsivo - Funciona perfeitamente em desktop, tablet e mobile.
.Interface Moderna - Design escuro com gradientes e animações suaves.
.Autoplay e Loop - Reprodução automática e contínua.
.Playlist Dinâmica - Lista de músicas gerada automaticamente dos arquivos MP3.
.Arquitetura MVC - Código organizado e fácil de manter a manutenção do código.
